🎯 À propos de ce dashboard

Cette visualisation interactive présente les données budgétaires réelles 2020 de la Communauté Urbaine Caen la Mer (CUCLM). Développée avec des technologies modernes, elle permet d'explorer facilement 10 communes avec leurs budgets détaillés représentant plus de 19 millions d'euros.

🛠️ Technologies

  • Python: Analyse de données (Pandas)
  • Leaflet: Cartographie interactive
  • Plotly: Graphiques dynamiques
  • JavaScript ES6+: Interactivité

🎯 Fonctionnalités

  • 🗺️ Carte interactive avec sélection
  • 📊 Graphiques dynamiques temps réel
  • 🔍 Filtrage par fonction et type
  • 📱 Interface entièrement responsive
